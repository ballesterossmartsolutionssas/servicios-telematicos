# Entrega Parcial 2 - Servicios Telematicos

Este branch contiene la entrega del segundo parcial de la materia.

## Ruta principal
- `parcial_2/`

## Contenido del parcial 2
- `parcial_2/README_PARCIAL2.md`: resumen general de la entrega.
- `parcial_2/GUIA_SUSTENTACION_PARCIAL2.md`: guia base para sustentar.
- `parcial_2/GUIA_SUSTENTACION_PARCIAL2_COMPLETA.pdf`: guia completa.
- `parcial_2/GUIA_CELULAR_PARCIAL2.pdf`: guia corta para consulta rapida.
- `parcial_2/GUIA_CELULAR_PARCIAL2_DETALLADA.pdf`: guia detallada para el celular.
- `parcial_2/ULTIMA_HORA_SUSTENTACION_PARCIAL2.txt`: comandos y respuestas rapidas.
- `parcial_2/PUNTO_2_DOT_CHANCUCO.txt`: guion rapido del punto 2.
- `parcial_2/PUNTO_3_SFTP_CHANCUCO.txt`: guion rapido del punto 3.

## Estructura tecnica
- `parcial_2/0_enunciado/`: PDF original del parcial.
- `parcial_2/0_certificados_base/`: CA y certificados usados en FTPS.
- `parcial_2/1_ftps_ufw/`: scripts y evidencia de FTPS con UFW.
- `parcial_2/2_dot/`: scripts y evidencia de DNS over TLS.
- `parcial_2/3_sftp_ufw/`: scripts y evidencia de SFTP con UFW.
- `parcial_2/4_comandos_sustentacion/`: comandos rapidos de apoyo.
- `parcial_2/5_evidencia_visual/`: capturas y pcaps para Wireshark.
